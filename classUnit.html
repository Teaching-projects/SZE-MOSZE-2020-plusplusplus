Detailed Description

It defines a RPG attacker Unit.

It contains the main properties and actions what the unit can do or could happen with it.

The Unit can duel with other units. The winner will be the unit which has more damage or more health, but the attack cooldown parameter can twist the result. The lower cooldown makes the Unit faster than the other.

Note
It contains recursive solutions
